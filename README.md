## Install

A mix of macos and linux dotfiles.

```sh
$ git clone git@github.com:hellozimi/dotfiles
$ cd dotfiles
$ sh symlink.sh
```

### Linux (ubunutu 20.10)

* bspwm & sxhkd
* Alacritty
* Polybar
* betterlockscreen (i3lock-color)
* neovim

### macOS

For specific macOS settings run

```sh
$ ./macos
```
